## 使用

### 方法

1. 开始界面：时间、温度、光强度循环显示，按开关2可以使时间初始化
2. 按开关1：只显示时间，按开关2可以使时间初始化
3. 按开关1：只显示温度
4. 按开关1：只显示光强度
5. 按开关1：调整时模式，按开关2：时加一
6. 按开关1：调整分模式，按开关2：分加一

### 说明

1. 起始时间设置的是0分0秒，这样也就相当于一个计时器；如果想要当时钟，就需要一直供电。
2. 本来是带有蜂鸣器的，但是感觉用处不大，就没有使用。
